This repository contains **multiple folders** representing different stages of development:


### Folder Explanation

- **backend/**  
  Contains the server-side code built with Express and MongoDB, including authentication using JWT.

- **frontend/**  
  Initial frontend created using **Create React App (CRA)**.

- **frontend-vite/**  
  The same CRA frontend migrated to **Vite** after CRA deprecation.

- **client/**  
  A fresh frontend rebuilt from scratch using **Vite + Tailwind CSS**, following better UI practices and clean component structure.
